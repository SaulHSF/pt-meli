# Documentacion tecnica

Este directorio centraliza la documentacion viva del proyecto.

## Estructura

- `architecture/`: vistas de arquitectura, capas, estructura y estrategia de testing.
- `adr/`: registros de decisiones arquitectonicas (Architecture Decision Records).
- `flows/`: flujos funcionales clave del producto.
- `deployment/`: notas de despliegue y entornos.
